Hacker News Alerts

Hacker News Alerts is an app developed by two hackers who wanted push notifications from news.ycombinator.com. However, it turns out a lot of things get posted to Hacker News. Getting push notifications for everything would be overwhelming and annoying. So Hacker News Alerts allows you to set a point threshold, and only receive push notifications after a story reaches the threshold you set.

If you want to get notified everytime a new JavaScript framework comes out, set your threshold to 100. If you want to get notified only when a vulnerability is discovered in Linux, set your threshold to 1000. The amount of notifications you recieve is up to you.
